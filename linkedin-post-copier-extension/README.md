# LinkedIn Post Copier (Chrome/Edge)

Minimalna wtyczka (Manifest V3), która dodaje przycisk **„Kopiuj”** do postów na LinkedIn i kopiuje treść posta do schowka.

## Instalacja (Chrome / Edge)

1. Wejdź w:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. Włącz **Tryb dewelopera**.
3. Kliknij **Załaduj rozpakowane** (Load unpacked).
4. Wskaż folder: `linkedin-post-copier-extension/` (ten katalog).

## Użycie

1. Otwórz LinkedIn (np. feed).
2. Przy poście pojawi się przycisk **Kopiuj** (prawy górny róg).
3. Kliknij **Kopiuj** → treść wyląduje w schowku.

## Ograniczenia (na razie)

- Kopiuje **widoczny tekst** (jeśli post jest zwinięty, kliknij najpierw „See more / Zobacz więcej”).
- LinkedIn często zmienia HTML/CSS, więc selektory są heurystyczne.
